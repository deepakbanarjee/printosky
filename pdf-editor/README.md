# PDF Flow Editor

Convert static PDFs into fluid, editable blocks.

## Setup

### Backend (Python)
1.  `cd backend`
2.  `pip install -r requirements.txt`
3.  `python main.py`

### Frontend (React + Vite)
1.  `cd frontend`
2.  `npm install`
3.  `npm run dev`

## Usage
1.  Open the web app.
2.  Drag and drop a PDF.
3.  Edit the text blocks. The layout adjusts automatically.
